## General Description
**CozyStays** is a website developed with the aim of facilitating the search and booking of local accommodation in Portugal, providing a simple, intuitive and effective experience, both for guests and hosts.

### Features

**User:**
- [x] Register a new account.
- [x] Log in and out.
- [x] Edit their profile, including their name, profile picture, password, and email.

**Freelancers:**
- [x] List new services, providing details such as category, pricing, delivery time, and service description, along with images.
- [x] Track and manage their offered services.
- [x] Respond to inquiries from clients regarding their services and provide custom offers if needed.
- [ ] Mark services as completed once delivered.

**Clients:**
- [x] Browse services using filters like category and location.
- [x] Engage with freelancers to ask questions or request custom orders.
- [x] Hire freelancers and proceed to checkout (simulate payment process).
- [x] Leave ratings and reviews for completed services.

**Admins:**
- [x] Elevate a user to admin status.
- [x] Introduce new service categories.
- [x] Oversee and ensure the smooth operation of the entire system.
